# Exclusive Crypto Community Landing Page

This repository contains the HTML and CSS files for my exclusive crypto community landing page.  
The page provides visitors with information about handpicked coin insights, strategies, and a direct link to our private Discord community.

## Features
- Handpicked coin recommendations
- Exclusive strategies for serious crypto enthusiasts
- Direct community access via Discord
- Modern, responsive design

## Usage
1. Upload the files to GitHub Pages
2. Update the Discord link in `index.html`
3. Your landing page is live at: https://GhostxJeko.github.io/krypto-community/
